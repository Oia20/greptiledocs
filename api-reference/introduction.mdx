## Base URL

The base URL for all API requests is `https://api.greptile.com/v2/`.

## Authentication

Use a Greptile API key for user identification and authentication. Include the API key in the request header: `Authorization: Bearer <API_KEY>`.

## Custom Headers

`X-GitHub-Token`: Include this header with your GitHub access token for operations requiring GitHub access: `X-GitHub-Token: <GITHUB_TOKEN>`.
